## 1. ¬øQu√© es una lista?

Una **lista** es un tipo de dato en Python que permite almacenar m√∫ltiples objetos. Es una **colecci√≥n ordenada y mutable** de elementos separados por comas y encerrados entre corchetes (`[]`).

```python
my_list = [1, None, True, "Soy una cadena", 256, 0]
```

## 2. Acceder y modificar elementos

Las listas permiten acceder a elementos individuales mediante √≠ndices (positivos o negativos), as√≠ como tambi√©n modificar sus valores.

```python
my_list = [1, None, True, 'Soy una cadena', 256, 0]

print(my_list[3])   # salida: Soy una cadena
print(my_list[-1])  # salida: 0 (√∫ltimo elemento)

my_list[1] = '?'
print(my_list)  # salida: [1, '?', True, 'Soy una cadena', 256, 0]

my_list.insert(0, "primero")   # inserta al inicio
my_list.append("√∫ltimo")       # agrega al final
print(my_list)
# salida: ['primero', 1, '?', True, 'Soy una cadena', 256, 0, '√∫ltimo']
```

## 3. Listas anidadas (listas dentro de listas)

Python permite **listas anidadas**, es decir, listas que contienen otras listas como elementos. Esto permite estructuras complejas como matrices o √°rboles.

```python
my_list = [1, 'a', ["lista", 64, [0, 1], False]]
```

## 4. Eliminar elementos o listas completas

Puedes eliminar elementos individuales usando `del`, o borrar toda la lista.

```python
my_list = [1, 2, 3, 4]
del my_list[2]
print(my_list)  # salida: [1, 2, 4]

del my_list  # elimina completamente la lista
```

## 5. Recorrer listas con bucles `for`

Las listas pueden recorrerse con un bucle `for` f√°cilmente.

```python
my_list = ["blanco", "morado", "azul", "amarillo", "verde"]

for color in my_list:
    print(color)
```

## 6. Obtener la longitud de una lista con `len()`

La funci√≥n `len()` permite saber cu√°ntos elementos hay en una lista.

```python
my_list = ["blanco", "morado", "azul", "amarillo", "verde"]
print(len(my_list))  # salida: 5

del my_list[2]
print(len(my_list))  # salida: 4
```

## 7. Funciones vs. M√©todos

- **Funciones**: se usan de forma general y reciben datos como argumentos.

  ```python
  longitud = len(my_list)
  ```

- **M√©todos**: son funciones asociadas a objetos, y se invocan sobre esos objetos.

  ```python
  my_list.append("nuevo")  # m√©todo llamado sobre una lista
  ```
