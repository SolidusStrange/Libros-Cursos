### 1. Tipos de bucles en Python

Python tiene dos tipos principales de bucles: `while` y `for`.

#### üîÅ Bucle `while`

El bucle `while` ejecuta una o m√°s instrucciones **mientras una condici√≥n booleana especificada sea verdadera**.

```python
# Ejemplo 1: bucle infinito
while True:
    print("Atrapado en un bucle infinito.")

# Ejemplo 2: cuenta regresiva
counter = 5
while counter > 2:
    print(counter)
    counter -= 1
```

#### üîÅ Bucle `for`

El bucle `for` ejecuta un bloque de c√≥digo muchas veces, y se usa para **iterar sobre una secuencia** (como una lista, diccionario, tupla, conjunto o cadena). Tambi√©n puedes usar `range()` para generar secuencias num√©ricas.

```python
# Ejemplo 1: recorrer una cadena
word = "Python"
for letter in word:
    print(letter, end="*")

# Ejemplo 2: imprimir solo n√∫meros pares
for i in range(1, 10):
    if i % 2 == 0:
        print(i)
```

### 2. Uso de `break` y `continue`

Estas instrucciones permiten modificar el flujo del bucle.

#### üîπ `break` ‚Äì salir del bucle

```python
text = "OpenEDG Python Institute"
for letter in text:
    if letter == "P":
        break
    print(letter, end="")
```

#### üîπ `continue` ‚Äì saltar a la siguiente iteraci√≥n

```python
text = "pyxpyxpyx"
for letter in text:
    if letter == "x":
        continue
    print(letter, end="")
```

### 3. Cl√°usula `else` en bucles

Tanto los bucles `while` como `for` pueden tener una cl√°usula `else`. Esta se ejecuta **cuando el bucle termina normalmente (sin `break`)**.

```python
n = 0

while n != 3:
    print(n)
    n += 1
else:
    print(n, "else")

print()

for i in range(0, 3):
    print(i)
else:
    print(i, "else")
```

### 4. La funci√≥n `range()`

La funci√≥n `range()` genera una secuencia de n√∫meros enteros y devuelve un objeto de tipo `range`. Su sintaxis es:

```python
range(inicio, fin, paso)
```

- `inicio` (opcional): valor inicial (por defecto es 0)
- `fin`: valor final (no se incluye)
- `paso` (opcional): incremento o decremento (por defecto es 1)

```python
# Ejemplo 1: rango simple
for i in range(3):
    print(i, end=" ")  # Salida: 0 1 2

# Ejemplo 2: contar hacia atr√°s
for i in range(6, 1, -2):
    print(i, end=" ")  # Salida: 6 4 2
```
