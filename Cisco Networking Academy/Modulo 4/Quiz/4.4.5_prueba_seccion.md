# 4.4.5 PRUEBA DE SECCIÓN

## Pregunta 1
**Código:**

```python
def message():
    alt = 1
    print("¡Hola, mundo!")
 
print(alt)
```

**Salida esperada:**  
Se arrojará una excepción `NameError`:

```
NameError: name 'alt' is not defined
```

## Pregunta 2
**Código:**

```python
a = 1
 
def fun():
    a = 2
    print(a)
 
fun()
print(a)
```

**Salida esperada:**

```
2
1
```

## Pregunta 3
**Código:**

```python
a = 1
 
def fun():
    global a
    a = 2
    print(a)
 
fun()
a = 3
print(a)
```

**Salida esperada:**

```
2
3
```
